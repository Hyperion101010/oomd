oomdA new userspace OOM killer

Use memory pressure thresholds for greater control during out-of-memory (OOM) events.

Get Started

Triggered by PSI memory pressure

Kill memory-hogging processes when PSI memory pressure exceeds a specified threshold.

Works with cgroup2

Works on a per-cgroup basis: kill an entire group of processes without compromising the system or the main workload.

More granular OOM-kill options

Deal more proactively and gracefully with increasing memory pressure by pausing some tasks, performing an app shutdown with a scheduled restart, or other specified actions.
